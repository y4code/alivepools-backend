# alivepools-backend

# Install
```shell
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

# Freeze dependencies
```shell
pip3 freeze > requirements.txt
```

# Run
```shell
flask --app alivepools-backend run
```

About Caddyfile
Just run command below to start caddy server in background
```
caddy start
```
